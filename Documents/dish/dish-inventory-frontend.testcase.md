# 菜品列表国际化测试用例

## 功能描述
在菜品列表中，新增菜品（新品上架）弹窗相关文案支持中英文切换。

## 测试环境
- **页面路径**: `/merchant/inventory`
- **功能入口**: 右上角新增按钮（+）

---

## 测试用例

### 测试用例 1: 新品上架弹窗标题国际化
**前置条件**:
- 用户已登录（BOSS/STAFF）
- 进入菜品列表页面

**操作步骤**:
1. 切换语言为中文
2. 点击右上角新增按钮打开弹窗
3. 关闭弹窗
4. 切换语言为英文
5. 再次打开弹窗

**预期结果**:
- 中文模式下弹窗标题为“新品上架”
- 英文模式下弹窗标题为“New Dish”

---

### 测试用例 2: 新品上架弹窗按钮国际化
**前置条件**:
- 用户已登录（BOSS/STAFF）
- 进入菜品列表页面

**操作步骤**:
1. 切换语言为中文
2. 打开新增弹窗
3. 观察底部按钮文案
4. 切换语言为英文
5. 再次打开弹窗

**预期结果**:
- 中文模式下按钮为“取消 / 确认上新”
- 英文模式下按钮为“Cancel / Confirm Add”

---

### 测试用例 3: 新品上架弹窗表单字段国际化
**前置条件**:
- 用户已登录（BOSS/STAFF）
- 进入菜品列表页面

**操作步骤**:
1. 切换语言为中文
2. 打开新增弹窗
3. 检查表单字段标签与占位符
4. 切换语言为英文
5. 再次打开弹窗并检查

**预期结果**:
- 中文模式下字段标签为“菜品名称 / 价格 / 分类 / 描述 / 标签 / 绑定食材”
- 英文模式下字段标签为“Dish Name / Price / Category / Description / Tags / Bind Ingredients”
- 占位符与提示语随语言切换同步变化

---

### 测试用例 4: 编辑菜品弹窗标题国际化
**前置条件**:
- 用户已登录（BOSS/STAFF）
- 进入菜品列表页面
- 存在可编辑的菜品

**操作步骤**:
1. 切换语言为中文
2. 点击某菜品的“编辑”进入弹窗
3. 关闭弹窗
4. 切换语言为英文
5. 再次进入编辑弹窗

**预期结果**:
- 中文模式下标题为“编辑菜品”
- 英文模式下标题为“Edit Dish”

---

## 测试验收标准
- ✅ 新增菜品弹窗标题、按钮、字段标签、占位符支持中英文切换
- ✅ 编辑菜品弹窗标题与按钮支持中英文切换
