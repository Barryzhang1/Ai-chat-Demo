# Ai-chat-Demo

## 概述
本项目 “Ai-chat-Demo” 是一个演示型的 AI 聊天应用，包含后端与前端两部分，旨在展示与测试基于模型的聊天交互与 UI 集成。

## 目录结构（概要）
- 后端: [chatBackEnd](chatBackEnd) — 聊天服务与模型调用接口
- 前端: [chatUi](chatUi) — 基于 React 的用户界面，使用 `webpack` 打包

## 功能亮点
- 实时对话界面与会话历史展示
- 前后端分离，便于替换或接入不同 AI 模型后端
- 简洁示例，便于教学与原型验证

## 技术栈
- 前端: React、Webpack、CSS
- 后端: 可接入任何支持的模型服务或自建 API（示例占位）

## 适用场景
- 原型展示、开发调试、前后端集成验证、教学演示

## 快速启动
- 启动前端（在 `chatUi` 中）:

```bash
cd chatUi
npm install
npm start
```

- 启动后端（视具体实现而定，示例）:

```bash
cd chatBackEnd
# 按后端说明安装依赖并启动，例如：
npm install
npm start
```
